# **App Name**: HostFlow Admin

## Core Features:

- Dashboard Overview: A centralized dashboard displaying key metrics like active listings, new bookings, check-in/out schedules, and recent activity.
- Listing Management: Tools to create, view, edit, and delete property listings, including details, availability, and pricing.
- Booking Management: Functionality to view, approve, decline, and modify bookings, track guest communications, and manage payment statuses.
- User Management: Access to view and manage host and guest profiles, including contact information and account activity.
- Automated Response Generator: An AI-powered tool to draft intelligent and personalized responses to common guest inquiries, check-in instructions, or local recommendations.
- Basic Reporting & Analytics: Simple reports on booking trends, occupancy rates, and revenue overview for business insights.

## Style Guidelines:

- Primary color: A sophisticated, professional blue (#2947A3) to convey trustworthiness and efficiency.
- Background color: A very light, muted blue-grey (#F0F1F5) providing a clean and easily readable canvas for data.
- Accent color: A vibrant teal-like blue-green (#1AE6B3) for interactive elements, highlights, and calls to action.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective appearance, ensuring excellent readability of data.
- Utilize clean, crisp, and easily recognizable line icons to represent different management sections and actions.
- Implement a modular, grid-based layout for dashboards and listing views, ensuring logical grouping of information and ease of navigation.
- Subtle and functional animations for data updates, tab transitions, and form submissions to provide user feedback without being distracting.
