# Manila Prime Property Management Admin

This is a Next.js 15 application optimized for native desktop deployment using **Tauri**.

## Getting Started
To run the web development server:
```bash
npm run dev
```

## Desktop Build
This project is pre-configured for **Static Export**. For instructions on how to build the native **Tauri** desktop application, please refer to:
- [docs/TAURI_BUILD.md](./docs/TAURI_BUILD.md)

## Key Features
- **Dashboard**: Real-time business intelligence with AI insights.
- **Unified Calendar**: Multi-unit availability tracking.
- **Financial Ledger**: Integrated payment and expense tracking.
- **AI Assistant**: Automated guest responses and listing generators via external Cloud Functions.
